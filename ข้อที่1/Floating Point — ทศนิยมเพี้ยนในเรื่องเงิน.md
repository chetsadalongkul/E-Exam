# จุดผิดพลาดที่ 1 : Floating Point Error

ปัญหา
โค้ดเดิมคำนวณเงินเดือนและค่าล่วงเวลาโดยใช้ชนิดข้อมูล Number ของ JavaScript ซึ่งเป็น Floating Point ทำให้เกิดปัญหาความคลาดเคลื่อนของทศนิยมได้

ตัวอย่าง
0.1 + 0.2

ผลลัพธ์
0.30000000000000004
ในระบบเงินเดือน ความคลาดเคลื่อนดังกล่าวอาจส่งผลให้ยอดเงินสุทธิไม่ถูกต้อง

วิธีแก้
แปลงหน่วยเงินจากบาทเป็นสตางค์และคำนวณด้วยจำนวนเต็ม (Integer) ก่อน จากนั้นค่อยแปลงกลับเป็นบาทเมื่อส่งผลลัพธ์
